# Hangman Game

A Python command-line implementation of the classic Hangman game.

The player guesses letters to uncover a hidden word within a limited
number of incorrect attempts.

## Features
- Random word selection
- ASCII-based hangman display
- Input validation
- Replay support
- Win/Lose detection

## Concepts Used
- Functions and loops
- Sets and dictionaries
- Input validation
- Game state management
- Control flow with nested loops

## How to Run
```bash
python hangman.py
